# 漫长的笑点
从前有座山，山里有座庙，庙里有个老和尚和一个小和尚。

有一天，小和尚闲着没事，就拉着老和尚说：“师父师父，你给我讲个笑话吧，我无聊得快长出蘑菇了。”

老和尚放下手里的木鱼，慢悠悠地说：“行，那我给你讲一个很长很长的笑话，你可得认真听，别走神，不然就听不到最关键的地方了。”

小和尚赶紧盘腿坐好，双手托着下巴，眼睛瞪得圆圆的：“师父你讲吧，我保证不走神！”

老和尚清了清嗓子，开始说道：“从前啊，在很远很远的地方，有一个小村庄，这个村庄坐落在一片一望无际的田野旁边，田野里种满了麦子、水稻、玉米，还有各种各样的蔬菜，春天的时候，田野里开满了油菜花，黄澄澄的一片，风一吹，就像金色的海浪一样翻滚，特别好看。”

小和尚点点头：“嗯，好看，师父然后呢？”

老和尚继续说：“这个村庄里住着几十户人家，大家都是种地的农民，每天日出而作，日落而息，日子过得平淡又安稳。村里有个姓张的老汉，大家都叫他张老汉，张老汉今年六十多岁了，头发胡子都白了大半，背有点驼，但是身体还算硬朗，每天还是会去田里忙活，他家里有一亩三分地，种的都是小麦，每年收成还不错，够自己家吃，还能剩下一些拿去镇上换点油盐酱醋。”

“张老汉有个老伴，姓李，大家叫她李婆婆，李婆婆比张老汉小两岁，手脚麻利，家里的家务活全包了，做饭、洗衣、喂鸡、喂鸭，样样都做得井井有条。他们家还养了一头老黄牛，这头牛跟着他们十几年了，脾气特别温顺，耕地拉车都靠它，张老汉对这头牛可好了，每天都给它割最嫩的青草，喂最饱满的粮食，冬天还会给牛棚铺厚厚的干草，生怕它冻着。”

小和尚打了个小小的哈欠，又立刻坐直：“师父，牛听话吗？”

“听话极了，”老和尚说，“这头牛从来不会乱跑，也不会顶撞人，村里的小孩有时候还会摸它的头，它都安安静静的。除了这头牛，他们家还养了三只母鸡，两只公鸡，每天早上公鸡一打鸣，全村人就知道该起床了。母鸡每天都会下蛋，有时候一天一个，有时候两天一个，攒够了十几个，李婆婆就会装在竹篮子里，拿到镇上的集市去卖，换几个零钱，给张老汉买袋旱烟，或者给自己买根缝衣针。”

“村庄旁边有一条小河，河水清清的，能看见水底的鹅卵石，还有小鱼小虾在水里游来游去。夏天的时候，村里的小孩都喜欢跑到河边玩水、摸鱼、捉螃蟹，河水不深，刚到小孩的膝盖，所以大人们也不用担心。河边长着很多柳树，柳枝长长的，垂到水面上，风一吹，柳枝就轻轻摆动，特别凉快。张老汉有时候干完农活，就会坐在柳树下歇脚，抽一袋旱烟，看着河水慢慢流走，觉得日子特别舒坦。”

“离村庄不远的地方，有一座小山，山上长满了松树、柏树，还有各种各样的野花，春天有迎春花、桃花，夏天有荷花、栀子花，秋天有菊花、桂花，冬天还有腊梅，一年四季都有花开。山上还有一条小路，能通到山顶，山顶上有一块大石头，站在石头上能看见整个村庄，还有远处的田野和河流。村里的人有时候会上山砍柴，或者采点野果子，比如野草莓、野山楂，味道酸酸甜甜的，特别好吃。”

小和尚揉了揉眼睛，小声问：“师父，笑话什么时候开始啊？”

老和尚摆了摆手：“别急别急，铺垫还没说完呢，这个笑话的精髓就在后面，你再耐心听听。”

小和尚只好又坐好，继续听。

“话说有一天，天气特别好，万里无云，太阳暖洋洋地照在大地上，张老汉像往常一样，牵着老黄牛去田里耕地。耕到中午的时候，太阳有点晒了，张老汉就把牛拴在田边的大树下，自己坐在树荫下休息，从怀里掏出李婆婆给他准备的午饭，是两个白面馒头，还有一小碟咸菜。张老汉啃着馒头，喝着随身携带的水，看着自己耕好的地，心里美滋滋的，想着今年肯定又是一个丰收年。”

“就在这个时候，从远处走来了一个货郎，货郎挑着担子，担子两头挂着很多小玩意儿，有小孩玩的拨浪鼓、小风车，有女人用的头绳、发卡，还有男人用的烟袋、打火机，还有各种糖果、零食。货郎一边走一边摇着拨浪鼓，‘咚咚咚’的声音在田野里特别响亮。”

“张老汉看见货郎，就喊住了他：‘货郎老弟，过来歇会儿吧！’”

“货郎听见喊声，就挑着担子走了过来，把担子放在树下，擦了擦汗说：‘多谢老哥了，这太阳确实有点晒。’”

“张老汉递给他一碗水：‘喝点水吧，解解渴。’”

“货郎接过水，一饮而尽，然后从担子里拿出一块糖，递给张老汉：‘老哥，尝尝这个水果糖，甜得很。’”

“张老汉接过糖，剥开糖纸放进嘴里，果然甜滋滋的，他笑着说：‘好久没吃过这么甜的糖了，还是你们货郎走南闯北，能吃到好东西。’”

“货郎笑着说：‘嗨，也就是混口饭吃，每天走村串户，累是累点，但是能见识到各地的风土人情，也挺有意思的。老哥，你这牛可真壮实，耕地肯定好使。’”

“张老汉摸了摸老黄牛的头：‘那可不，这牛跟我十几年了，比我家小子还听话呢。’”

“两人就这么你一言我一语地聊了起来，货郎跟张老汉讲他在别的村庄看到的新鲜事，比如哪个村有人家娶媳妇，办了特别热闹的酒席，哪个村的果园大丰收，果子多得卖不完，哪个村来了个戏班子，唱了好几天的大戏。张老汉听得津津有味，觉得外面的世界真有意思。”

“聊了大概半个时辰，货郎要继续赶路了，就挑起担子，跟张老汉道别：‘老哥，我先走了，下次再过来跟你聊天。’”

“张老汉挥挥手：‘慢走啊，路上注意安全！’”

“货郎走了之后，张老汉又歇了一会儿，就牵着牛继续耕地，耕到太阳快落山的时候，地也耕得差不多了，他就牵着牛往家走。回到家的时候，李婆婆已经做好了晚饭，是玉米粥，还有炒青菜，还有几个鸡蛋，是早上母鸡刚下的。”

“吃完饭，天也黑了，张老汉和李婆婆就坐在院子里乘凉，村里的邻居也都出来了，大家聚在一起聊天，东家说说自家的庄稼，西家讲讲自家的孩子，气氛特别热闹。聊到半夜，大家都困了，就各自回家睡觉了。”

“第二天一早，公鸡打鸣，张老汉起床，喂了牛，喂了鸡，然后又准备去田里忙活。刚走到门口，就看见村里的王大叔急匆匆地跑过来，王大叔跑得气喘吁吁，脸上还带着着急的神色。”

“张老汉赶紧问：‘老王，咋了这是？出什么事了？’”

“王大叔喘着气说：‘老张，不好了，村东头的老刘家，他家的猪圈塌了，猪都跑出来了，到处乱窜，大家都去帮忙赶猪了，我来叫你一起去！’”

“张老汉一听，赶紧放下手里的农具：‘行，我这就跟你去！’”

“两人一路跑到村东头，果然看见老刘家的猪圈塌了一大半，好几头肥猪在村里的路上乱跑，有的往田里跑，有的往河边跑，还有的撞翻了路边的菜篮子。老刘和他的儿子正满头大汗地追猪，旁边已经围了不少村民，大家都在帮忙赶猪。”

“张老汉也赶紧加入进去，大家围追堵截，费了好大的劲，才把跑出来的猪全都赶回了临时搭的围栏里。老刘感激地对大家说：‘真是太谢谢各位乡亲了，要是猪跑丢了，我这一年的心血就白费了。’”

“村民们都说：‘乡里乡亲的，客气啥，互相帮忙是应该的。’”

“折腾了一上午，大家都累坏了，老刘非要留大家吃饭，大家推辞不过，就留下来吃了顿便饭。吃完饭，张老汉才回到自己家，继续干自己的活。”

“日子就这么一天一天过着，平淡又安稳，转眼就到了秋天，田野里的麦子都成熟了，金黄金黄的，一眼望不到边。张老汉开始忙着收割麦子，每天天不亮就起床，带着镰刀去田里，一直忙到天黑才回家。李婆婆也会去田里帮忙，割麦子、捆麦子，虽然累，但是看着满田的麦子，心里都是欢喜。”

“收完麦子，还要晒麦子，张老汉把麦子铺在自家的晒谷场上，每天翻晒几遍，等麦子晒干了，就装进粮仓里。今年的收成比往年还好，粮仓都快装满了，张老汉和李婆婆笑得合不拢嘴。”

“晒麦子的时候，村里的小孩经常会跑到晒谷场上玩，有的追逐打闹，有的在麦堆上打滚，还有的捡掉在地上的麦穗，拿回家让大人煮着吃。张老汉也不生气，只是笑着提醒他们：‘慢点跑，别摔着了，别把麦子踩坏了。’”

“有一天，晒谷场上来了一只小麻雀，小麻雀落在麦堆上，偷偷啄食麦粒。张老汉看见了，也没赶它，只是说：‘吃吧吃吧，就当给你加餐了。’小麻雀好像听懂了一样，啄了几口，就飞到旁边的树上，叽叽喳喳地叫着，好像在道谢。”

“秋天过后，就是冬天了，天气越来越冷，刮起了北风，还下起了小雪。田里的农活都忙完了，村民们就待在家里，烤着火，聊着天，做着手工活。张老汉也不用去田里了，每天就在家里喂喂牛，喂喂鸡，跟李婆婆说说话，日子过得悠闲自在。”

“有一天下雪，雪下得特别大，地上积了厚厚的一层，足有半尺深。村里的路都被雪盖住了，大家都待在家里不出门。张老汉坐在屋里，烤着柴火，看着窗外的雪景，突然想起了小时候的事。”

“他小时候，也是住在这个村庄里，那时候家里穷，吃不饱穿不暖，冬天经常冻得手脚发紫。但是那时候快乐也很简单，和小伙伴们一起在雪地里堆雪人、打雪仗，就能开心一整天。那时候村里的树比现在还多，河水比现在还清，日子虽然苦，但是心里很踏实。”

“想着想着，张老汉就笑了，李婆婆看见他笑，就问：‘老头子，你笑啥呢？’”

“张老汉说：‘我想起小时候下雪天打雪仗的事了，那时候真有意思。’”

“李婆婆也笑着说：‘可不是嘛，那时候我们也一起玩过，你还把雪塞进我脖子里，冻得我直哆嗦。’”

“两人你一言我一语，回忆着年轻时候的事，脸上都洋溢着幸福的笑容。”

小和尚这时候已经困得不行了，脑袋一点一点的，他迷迷糊糊地问：“师父……笑话……到底讲完了没有啊……”

老和尚看着昏昏欲睡的小和尚，终于慢悠悠地说出了最后一句话：

“讲完了。”

小和尚猛地一激灵，睁开眼睛：“啊？这就完了？笑点在哪啊？”

老和尚捧着肚子哈哈大笑：

“**笑点就是——我跟你扯了这么久废话，你居然真的认认真真听到了最后！**”
